🚗 CarShow Live

Real-time interactive map for car show events — vendors, sponsors, VIPs, heat map, poker run, and more.

Tech Stack

  • Express.js backend (single-file server)
  • Leaflet.js + OpenStreetMap (free tiles, no API key)
  • In-memory state with JSON file persistence
  • Single HTML file frontend with vanilla JS (no framework)
  • Docker deployment on NAS

Features 90% Complete

  • ✅ Role-based dashboards (Admin, Organizer, Vendor, Sponsor, VIP, Registered User, User)
  • ✅ Interactive map with heat signatures, day/night tiles

  • ✅ Pin dropping (Spot, Car Spot, Bomb Food) with countdown timers
  • ✅ Pin likes (one per user) with ❤️ count
  • ✅ User management system with mini-map location picker
  • ✅ Pin management system with CRUD
  • ✅ Community directory with search, car photos, Instagram

Deployment

  • Local: Docker Compose on UGREEN NAS — http://192.168.1.44:3000
  • Public: Cloudflare tunnel via cloudflared in Docker start script
  • Data: Volume mount /volume1/docker/1/carshow-data:/data
